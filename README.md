
# Decode Morse code

The Decode Morse code accepts Morse code as string and decodes. 

### Cloning the project

 git clone https://github.com/sentayhu19/Decode-Morse-code.git <Your-Build-Directory>
``` 
- cd Decode-Morse-code
```
## Prerequisites

Vscode or anyother
Setup

    Ruby
## Run tests
    Run bundle install
    Run rubocop
